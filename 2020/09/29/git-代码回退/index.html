git 代码回退

代码回退

默认参数 -soft,所有commit的修改都会退回到git缓冲区

参数–hard，所有commit的修改直接丢弃

$ git reset –hard HEAD^ 回退到上个版本 $ git reset –hard commit_id 退到/进到 指定commit_id

推送到远程

$ git push origin HEAD –force
