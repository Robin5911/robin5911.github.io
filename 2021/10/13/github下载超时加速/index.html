github下载超时加速

国内访问github太慢，甚至超时，咋办？

第一步：找 IP
首先找到 github.com 的 IP 地址，可以在 IPAddress 网站上查找。还可以添加 github.global.ssl.fastly.net 的映射，这个有助于加速。

注意：隔段时间地址会变，网上找的有些都过期了，建议还是自己去查询一下。

第二步：修改 host
vim /etc/hosts
将找到的最新地址映射添加到 hosts 中。

亲测,确实有效果噢~
